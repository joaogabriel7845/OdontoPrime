# 🦷 OdontoPrime — Landing Page

Landing page profissional para clínica odontológica, desenvolvida com foco em conversão e agendamento via WhatsApp.

## 🖥️ Preview

> Deploy: [odonto-prime-eight.vercel.app](https://odonto-prime-eight.vercel.app/)

---

## 🚀 Tecnologias

- [React](https://react.dev/)
- [Vite](https://vitejs.dev/)
- [Tailwind CSS v4](https://tailwindcss.com/)
- [Framer Motion](https://www.framer.com/motion/)
- [Font Awesome](https://fontawesome.com/)
- [@smastrom/react-rating](https://github.com/smastrom/react-rating)

---

## 📋 Seções

- **Header** — Logo, navegação responsiva e botão de agendamento. Menu drawer no mobile.
- **Hero** — Título principal, subtítulo e CTAs de agendamento.
- **Estatísticas** — Números de impacto da clínica.
- **Serviços** — Cards com ícones coloridos para cada tratamento.
- **Sobre** — História da clínica e diferenciais com lista de qualidades.
- **Benefícios** — Por que escolher a OdontoPrime.
- **Depoimentos** — Cards com avaliação em estrelas e avatar gerado dinamicamente.
- **FAQ** — Accordion com animação de abrir/fechar.
- **CTA** — Chamada final com botão direto para o WhatsApp.
- **Footer** — Endereço, telefone, redes sociais e copyright.

---

## ✨ Funcionalidades

- Scroll suave entre seções via navegação
- Animações de entrada ao rolar a página (Framer Motion)
- Menu drawer animado no mobile com overlay
- Accordion interativo no FAQ com estado React
- Avatares gerados via [UI Avatars](https://ui-avatars.com/)
- Layout totalmente responsivo (mobile first)
